## Sqoop 이란

[이미지]()

Sqoop (SQL to Hadoop)은 관계형 데이터베이스(RDB)와 분산 파일 시스템(HDFS) 사이의 양방향 데이터 전송을 위해 개발된 툴이다.

> Sqoop은 정형 데이터를, Kafka와 Flume은 비정형 데이터를 다룬다.
