## Airflow란?
Python 코드로 워크플로를 작성하고, 스케줄링, 모니터링하는 플랫폼이다. 이를 이용해 Data Engineering의 ETL 작업을 자동화하고, DAG (Directed Acyclic Graph) 형태의 워크플로 작성이 가능하다.
